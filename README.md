# Simple city app

Technology stack:

* Java 11/17 + Spring Boot 
* React
* Graphql

## Running with Java 11
```
mvn clean verify
```
and then 
```
java -jar target/cities-0.0.1-SNAPSHOT.jar
```

Open `localhost:8080` and have fun

## Running with Java 17
```
mvn clean verify -P j17
```
and then 
```
java -jar target/cities-0.0.1-SNAPSHOT.jar
```

Open `localhost:8080` and have fun.
